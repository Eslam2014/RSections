Slidify

This repo contains examples of presentations created using Slidify. Note that you need to install the dev versions of Slidify and SlidifyLibraries to be able to slidify these presentations.

  1. Google IO2012
  2. DeckJS
  3. RevealJS
  4. FlowtimeJS
  5. ImpressJS
  6. Landslide
  7. DZSlides
  8. Shower
  9. Showoff
  10. FathomJS
  11. HTML5Slides
  12. Slideous
